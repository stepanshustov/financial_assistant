Финансовый помошник.
Приложение разработанное для учёта и контроля доходов и расходов пользователя, а так же для удобного просмотра оных.

Проект выполнил Шустов Степан 15 лет, г. Ульяновск
ученик Академии Яндекс Лицея

Используемые невстроенные библиотеки:
1) PyQt5 5.15.10 ( pip install PyQt5)
2) pyhtml2pdf 0.0.7 (pip install pyhtml2pdf)
